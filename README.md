# MEAN Stack App (Angular 1.x)

Prerequisites

  - [NodeJS]
  - [MongoDB]
  - [ExressJS]
  - [AngularJS]

Installation
  - Download the repository
  - Open cmd 
  - Install npm modules: 
```sh
 npm install
```
  - Start up the server: 
```sh
 node server.js
```
  - View in browser at http://localhost:8080
